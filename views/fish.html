Each of the fish in our simulation has a "chromosome." In biology, this chromosome would be a series of nitrogen bases. While some other genetic algorithms encode characteristics in binary and represent the chromosome as a series of ones and zeros, we decided to simplify our model. Each fish's chromosome is an array of __ numerical values that control various aspects of its appearance and behavior. A typical chromosome may look like this:

[50, 20, 5, 30, 2, 0, 100, 50, 240, 100, 50]

The first five values represent the dimensions of the fish, in this order:
1: Body Length
2: Body Width
3: Tail Length
4: Tail Width
5: Eye Radius

All of the dimensional values are measured in pixels, so the example chromosome above would have a body of length 50px and width 20px, a tail of length 50px and width 30px, and an eye of radius 3px. This fish would look like this:
